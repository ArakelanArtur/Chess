# ♟️ Browser Chess Game

Веб-приложение для игры в шахматы, написанное на чистом JavaScript без использования сторонних библиотек и фреймворков. 

[**🚀 Запустить игру в браузере (Live Demo)**](https://<ВАШ_НИК>.github.io/<ИМЯ_РЕПОЗИТОРИЯ>/)

## 🔥 Ключевые особенности

*   **⚡ Умный индикатор хода:** Интерфейс динамически отображает, чей сейчас ход (**Белые** / **Черные**).
*   ** Vanilla JS:** Вся логика, анимации и валидация ходов написаны на чистом JavaScript.
*   **🎨 Современный CSS:** Чистая разметка доски, адаптивные стили и подсветка элементов.
*   **🖱️ Удобное управление:** Поддержка кликов для выбора и перемещения фигур по клеткам.

---

## 🛠️ Стек технологий

*   **HTML5** — семантическая структура игрового поля.
*   **CSS3** — стилизация шахматной доски, фигур и панели состояния.
*   **JavaScript (ES6+)** — логика игры, переключение ходов и обработка событий.
